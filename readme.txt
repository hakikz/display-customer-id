Display customer id for logged in users, using [dci_customer_id] shortcode. Attributes can be passed as title. For example: [dci_customer_id title="User ID: "]

**Note: The customer ID will be displayed for the logged in customers only.**

== Installation ==
1. Upload the unzipped folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
